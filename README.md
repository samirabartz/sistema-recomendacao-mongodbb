# Sistema de Recomendação com MongoDB

Case Study — Administração da Informação
Universidade Europeia

## Descrição

Sistema de recomendação para uma plataforma de e-commerce implementado com MongoDB, utilizando modelação de dados, índices, agregações e sharding.

## Estrutura do Projeto

* 1_criacao_colecoes_indices.js — Criação das coleções e índices
* 2_insercao_dados_exemplo.js — Inserção de dados de exemplo
* 3_algoritmo_recomendacao.js — Algoritmo de recomendação
* 4_configuracao_sharding.js — Configuração de sharding
